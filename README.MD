# Infinity Make

#### 一个极简化的高扩展与高性能构建系统

*有多简单？只要一个单脚本文件！*

###### 注：目前处于极早期阶段，尚未完成大部分功能！

#### 使用方法
##### 于pwsh环境下进行
##### 1.clone本项目
```shell
git clone https://github.com/yinbailiang/infinity_make.git
```
##### 2.构建本项目
```shell
Set-Location infinity_make
./build.ps1
```
构建结果为生成于当前目录下的infinity_make.ps1
##### 3.使用本项目
*对不起，目前项目处于极早期阶段，API等都未稳定，所以不建议使用。*
